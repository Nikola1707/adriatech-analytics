Project Overview
AdriaTech d.o.o is an excel-based data analysis project focused on analyzing business data of a fictional company called "AdriaTech".  
The project demonstrates data cleaning, analysis, and visualization skills using structured tables, pivot tables, and dashboard.

This project is created as a portfolio example for data analytics / business analytics roles.

---

Tools & Technologies
- Microsoft Excel  
- Pivot Tables  
- Data aggregation and analysis  
- Charts and dashboards  

---

File Structure
The Excel file contains the following sheets:
- Data – original dataset    
- Dashboard – visual representation of key metrics  

---

  Project Goals
- Analyze business performance data  
- Identify trends and patterns  
- Present insights in a clear and visual format  

---

  Key Insights
- Overview of overall business performance  
- Comparison between key categories  
- Identification of best and worst performing segments  

---

  Future Improvements
- Automate data processing using Power Query or Python  
- Expand the dashboard with additional KPIs  
- Add more time periods for deeper analysis  
- Migrate the project to Power BI or Tableau
